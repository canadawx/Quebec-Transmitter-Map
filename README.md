# Quebec-Transmitter-Map

This is the Quebec Transmitter Map, this map shows you every transmitter like FM/TV/AM/WXR and SW! This data comes from ISED public data about licensed transmitters around the Quebec province. This map is currently W.I.P (Work.In.Progress) so issues can occur most of the times. This map is automatically updated every few mins so if new transmitters have been licensed, it will update and add the new transmitter. If issues occur and you want to report it to me, e-mail chzmteam@hotmail.com.
