# The-Canadian-Transmitter-Map

This is The Canadian Transmitter Map, this map shows you every transmitter like FM/TV and AM! (WXR is coming soon). This data comes from ISED public data about licensed transmitters around the Quebec province. This map is synced by the user, so if new transmitters have been licensed, you can click on sync and it will add the new transmitters. If issues occur and you want to report it to me, e-mail chzmteam@hotmail.com.
